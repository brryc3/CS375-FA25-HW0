Try info stack. Explain what you see.
info stack shows the current stack frames, including function calls and their arguments. Since wc.c is simple, you may only see main.

Try info frame. Explain what you see.
info frame shows details about the current stack frame, such as the function, arguments, saved registers, and source file location.

Try info registers. Which registers are holding aspects of the program that you recognize?
Registers like rbx hold argv and r12 holds argc.
